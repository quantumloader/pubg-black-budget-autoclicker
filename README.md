# PUBG Black Budget AutoClicker (ENG)

A simple HTML script that automatically clicks the **Claim Drops Rewards** button in PUBG web events. It handles server conflicts (HTTP 409), error modals, and automatically resumes clicking until a **Congratulations / Activate** message appears.

## Features

- Automatically clicks **Claim Drops Rewards**.
- Handles error popups with **OK** buttons.
- Detects server `409 Conflict` responses and waits for the correct timing.
- Stops automatically once **Congratulations / Activate** appears.
- Lightweight, runs directly in the browser.
- Simple GUI with Start / Stop / Reset buttons.

## Usage

1. Copy script from `script.html` and paste to console in your browser while logged in to PUBG Black Budget website.  
2. Click **Start** on the GUI to begin auto-clicking.  
3. Click **Stop** to pause the script.  
4. Click **Reset** to clear flags if needed.  

> ⚠️ Use responsibly. Frequent clicks may trigger server conflicts; the script automatically handles them.

# PUBG Black Budget AutoClicker (RU)

Простой HTML скрипт, который автоматически нажимает кнопку **Claim Drops Rewards** на веб-мероприятиях PUBG. Обрабатывает серверные конфликты (HTTP 409), модальные окна с ошибками и продолжает работу до появления сообщения **Congratulations / Activate**.

## Возможности

- Автоматическое нажатие **Claim Drops Rewards**.  
- Обработка всплывающих ошибок с кнопкой **OK**.  
- Детектирование серверных ответов `409 Conflict` и ожидание корректного времени для клика.  
- Авто-остановка при появлении **Congratulations / Activate**.  
- Лёгкий скрипт, работает прямо в браузере.  
- Простой GUI с кнопками Start / Stop / Reset.

## Использование

1. Открыть `script.html` скпоировать код, будучи авторизованным на событии PUBG Black Budget вставить скрипт в консоль.  
2. Нажать **Start** на GUI для запуска автокликера.  
3. Нажать **Stop**, чтобы приостановить скрипт.  
4. Нажать **Reset**, чтобы очистить флаги и подготовиться к новой попытке.  

> ⚠️ Используйте ответственно. Частые клики могут вызвать серверные конфликты; скрипт автоматически их обрабатывает.
